Installation & Usage

üì¶ NPM Installation

Recommended to install via npm for latest version and type definitions:

üöÄ Basic Usage

Import as ES6 module and initialize:

import { tabScroller } from 'tab-scroller'; const tabs = new tabScroller('.my-tabs', { spaceBetween: 15, clickToActive: true, clickToScroll: true });
